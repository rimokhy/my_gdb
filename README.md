# my_gdb

Hello.
Here's a pre-alpha gdb version made in order to learn how a debugger works.
Supports only x86_64.
Made in C++

Commands :

disassemble <label> : Disassembly
run <args>
brk *<0xaddr> : Breakpoints

    -> continue : Program continues running

    -> print <register_name>

    -> set <register_name
